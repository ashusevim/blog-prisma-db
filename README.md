# Blog DB with Migration + Rollback

This repository shows how to manage database migrations and rollbacks using Prisma ORM. 

It includes:

- User management, Blog post creation and Comments with proper indexing
- Migration history tracking
- Schema definition with Prisma

Tech stack: Typescript, Expressjs, Prisma, PostgreSQL

Key features:
- Database schema versioning
- Migration rollback capability
- Proper relationship modeling
- Index optimization for performance

This is a learning project to understand database migration patterns
